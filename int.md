## 섹션 1. 시스템 소개 헤더

**분당발전본부 위험성평가 지원 시스템**

- 소속 : 행복플러스 분당발전본부 안전관리실 (실장 :)
- 목적 : 건설현장 시공사 위험성평가 검토 지원
- 데이터 : 산업재해 고위험요인 (2013~2021년) 사망/중상 재해사례 **2,574건**
- 출처 : 한국산업안전보건공단 SIF(Serious Injuries and Fatalities) 자료

> 로고 이미지(`images/logo.png`)와 함께 히어로 영역으로 구성

## 섹션 2. 제작 배경

건설안전 담당자가 시공사가 제출한 위험성평가를 검토하는 과정에서 느낀 한계에서 출발하였습니다.
      
### 문제 인식
- 제작자는 **전기공학 전공자**로서, 토목·건축·기계·전기를 아우르는 건설공사 전반에 대한 현장 경험이 부족
- 위험성평가를 검토하기 위해서는:
  1. **공정**을 알아야 하고
  2. 해당 공정에서 어떤 **기인물**(위험원)이 있는지 알아야 하고
  3. 기인물별로 어떤 **재해형태**가 발생하는지 알아야 함
- 그러나 이 세 가지를 체계적으로 정리한 자료가 없어 검토에 한계를 느꼈음

### 해결의 실마리: SIF 자료
- 한국산업안전보건공단의 **SIF(중대재해 고위험요인) 자료**를 알게 됨
- SIF 자료에는 다음이 체계적으로 정의되어 있었음:
  - 건설공사 **전체 공정**의 분류 체계 (12개 공종 → 49개 작업명 → 113개 단위작업명)
  - 공정별 **주요 기인물** 정의 (58개 기인물, 32개 기인물분류)
  - 기인물별 **재해형태** 정의 (20개 재해형태)
- 이를 웹 시스템으로 재구성하여 누구나 쉽게 활용할 수 있도록 개발

## 섹션 3. 활용 방법

### 기존 SIF 자료의 한계
- PDF와 매크로 기반 Excel로 배포 → 현장에서 즉시 활용이 어려움
- 특정 공정이나 기인물을 조회하려면 복잡한 탐색 과정이 필요

### 본 시스템의 활용
- **웹 기반**: PC, 태블릿, 스마트폰에서 즉시 접근 가능
- **다차원 조회**: 공정 → 기인물 → 재해형태를 다양한 방향으로 조회
- **엑셀 다운로드**: 모든 재해사례 테이블을 엑셀(CSV)로 다운로드 가능
- **TBM/위험성평가 활용**: 다운로드한 재해사례를 TBM 자료나 위험성평가 후면에 덧붙여 활용
- **인쇄/PDF 지원**: 현재 화면을 그대로 인쇄하거나 PDF로 저장 가능
- **안전교육 강화**: 구체적인 재해사례와 감소대책을 통해 실효적인 안전대책 수립 및 이행 유도

## 섹션 4. 페이지별 기능 안내

> 6개 카드 그리드(2×3)로 구성, 각 카드 클릭 시 해당 페이지로 이동

### 4-1. 대시보드 (`index.html`)
- **아이콘**: home
- **주요 기능**:
  - 총 재해건수·고위험공종·최다재해형태 요약 카드
  - 고위험공종 / 최다재해형태 클릭 시 드롭다운으로 다빈도 순 조회 → 차트 즉시 반영
  - 공종별 재해건수 바 차트 (클릭 → 드릴다운: 작업명 → 단위작업명 → 재해사례)
  - 기인물 Top 10, 공종×재해형태 히트맵
- **활용**: 전체적인 재해 현황 파악, 고위험 영역 빠른 식별

### 4-2. 공정조회 (`pages/process-inquiry.html`)
- **아이콘**: search
- **주요 기능**:
  - 발전소공정 → 공종 → 작업명 → 단위작업명 → 기인물 **5단계 Cascading 필터**
  - 상위 필터 선택 시 하위 필터 자동 범위 제한
  - 기인물별 사고건수 차트 + 재해형태 비율 도넛 차트
  - 재해사례 목록 테이블 (컬럼 필터 + 엑셀 다운로드)
- **활용**: "이 공정에서 어떤 사고가 나는가?" 정방향 조회, 위험성평가 작성 시 참고

### 4-3. 기인물조회 (`pages/cause-inquiry.html`)
- **아이콘**: alert (경고삼각형)
- **주요 기능**:
  - 작업유형(고소/중장비/굴착/전기/중량물/화기/밀폐 등) → 12대 기인물 → 기인물 **역방향 필터**
  - 선택 기인물의 공종별·재해형태별·작업명별 분포 차트
  - 기인물 요약 (총 건수, 전체 순위, 12대 기인물 여부)
- **활용**: "이 기인물이 어느 공정에서 위험한가?" 역방향 조회

### 4-4. 재해형태 분석 (`pages/disaster-type.html`)
- **아이콘**: flame (불꽃)
- **주요 기능**:
  - 재해형태·기인물·공종·작업명·단위작업명 **다차원 필터**
  - 재해형태별 건수, 기인물별 건수, 공종별 건수 차트
  - 연계 필터: 선택 조합에 따라 나머지 필터 옵션 자동 제한
- **활용**: 특정 재해형태의 원인·공정 심층 분석

### 4-5. 위험성평가 (`pages/risk-assessment.html`)
- **아이콘**: shield (방패)
- **주요 기능**:
  - **3단계 카드 위자드**: 공종 선택 → 작업명 선택 → 단위작업명 선택
  - 각 단계에서 즉시 재해 건수 표시
  - 주요 위험요인 Top 10 카드 (기인물 + 재해형태 조합)
  - 재해사례 + 감소대책 통합 목록
  - **인쇄 지원**: 결과를 그대로 인쇄하여 현장 게시 가능
- **활용**: 현장 근로자용 위험성평가 수행 지원, TBM 자료 출력

### 4-6. 통계분석 (`pages/statistics.html`)
- **아이콘**: chart (차트)
- **주요 기능**:
  - **6개 탭**: 공종별 / 작업명별 / 단위작업명별 / 기인물별 / 재해형태별 / 교차분석
  - 각 탭별 차트 + 테이블 토글
  - 스택드 바 차트, 히트맵, 상세 분석 테이블
  - 엑셀 다운로드
- **활용**: 심층 통계 데이터 분석, 보고서 작성 참고 자료
