# SIF 프로그램 - 건설재해 위험성평가 차트 웹사이트 Roadmap

## 프로젝트 개요

**목적:** KOEN(한국남동발전) 건설현장 근로자를 위한 위험성평가 데이터 시각화 웹사이트

- Excel 기반 공정조회/기인물조회 대시보드를 웹사이트로 전환
- CSV 데이터를 활용한 필터별 차트 서비스 제공
- 기술스택: HTML, CSS, JavaScript, TailwindCSS (정적 웹사이트, 서버 불필요)

## 데이터 현황

### 핵심 데이터 (Excel DB시트 기준)
- **총 2,574건**의 건설현장 사망/중상 재해 사례
- 12개 공종 > 49개 작업명 > 113개 단위작업명 (3단계 계층구조)
- 57개 기인물, 12대 기인물 분류 체계
- 재해형태: 떨어짐(62.5%), 깔림뒤집힘(9.6%), 맞음(6.6%) 등

### CSV 파일 (10개) - 사전 집계 데이터
| 파일명 | 분석 축 | 주요 컬럼 |
|--------|---------|-----------|

| 작업명별_기인물_전체분석_정렬변경 | 공종 > 작업명 > 기인물 | 사고건수, 비중(%) |

| 기인물별_재해형태_전체분석 | 기인물 > 재해형태 | 사고건수, 비중(%) |

## Phase 1: 프로젝트 기반 구축

### 1.1 프로젝트 구조 생성
```
sif_Program/
├── index.html                  # 메인 페이지 (대시보드)
├── pages/
│   ├── process-inquiry.html    # 공정조회 (Excel ③공정조회 시트 대응)
│   ├── cause-inquiry.html      # 기인물조회 (Excel ③기인물조회 시트 대응)
│   ├── statistics.html         # 통계분석 (CSV 기반 차트)
│   └── risk-assessment.html    # 위험성평가 지원
├── css/
│   └── style.css               # 커스텀 스타일
├── js/
│   ├── app.js                  # 앱 초기화, 라우팅
│   ├── data-loader.js          # CSV/JSON 데이터 로딩
│   ├── chart-manager.js        # 차트 생성/업데이트 관리
│   ├── filter-manager.js       # 필터 상태 관리
│   └── utils.js                # 유틸리티 함수
├── data/
│   ├── db.json                 # Excel DB시트 → JSON 변환
│   ├── dropdown-ref.json       # 드롭다운참조 시트 → JSON 변환
│   └── csv/                    # CSV 파일 복사 (또는 JSON 변환)
├── source/                     # 원본 데이터 (기존)
└── Roadmap.md
```

### 1.2 데이터 전처리
- [ ] Excel DB시트(2,574건) → `data/db.json` 변환 (Python 스크립트)
- [ ] 드롭다운참조 시트 → `data/dropdown-ref.json` 변환 (계층구조 보존)
- [ ] CSV 파일 → `data/csv/` 복사 또는 JSON 변환
- [ ] 데이터 정합성 검증

### 1.3 기본 레이아웃 구축
- [ ] TailwindCSS CDN 연결
- [ ] 공통 레이아웃: 사이드바 네비게이션 + 메인 콘텐츠 영역
- [ ] 반응형 디자인 (모바일/태블릿/데스크톱)
- [ ] 다크모드 기본 적용 (건설현장 가독성)

**라이브러리 선정:**
- 차트: [Chart.js](https://www.chartjs.org/) (CDN, 경량, 반응형)
- 테이블: 자체 구현 (필터+정렬+페이지네이션)
- CSV 파싱: [PapaParse](https://www.papaparse.com/) (CDN)
- 아이콘: [Heroicons](https://heroicons.com/) 또는 inline SVG

## Phase 3: 공정조회 페이지 (Excel ③공정조회 대응)

> Excel의 순차 필터 방식을 웹 인터랙션으로 전환

### 3.1 계층형 필터 (Cascading Filter)
- [ ] 1단계: KOEN 공정 여부 선택
- [ ] 2단계: 공종 선택 (드롭다운)
- [ ] 3단계: 작업명 선택 (상위 선택에 따라 동적 생성)
- [ ] 4단계: 단위작업명 선택
- [ ] 5단계: 기인물 선택

### 3.2 조회 결과 영역
- [ ] **테이블**: 필터 조건에 맞는 재해사례 목록
  - 컬럼: No, 기인물, 재해형태, 재해개요, 재해유발요인, 위험성감소대책
  - 정렬/페이지네이션 지원
- [ ] **바 차트**: 선택된 조건의 기인물별 사고건수
- [ ] **바 차트**: 선택된 조건의 재해형태별 건수
- [ ] **파이 차트**: 선택된 조건의 재해형태 비율

### 3.3 사이드 패널
- [ ] 해당 공종의 작업명별 재해건수 순위
- [ ] 해당 공종의 기인물별 재해건수 순위

## Phase 5: 통계분석 페이지 (CSV 데이터 기반)

> CSV 파일들을 활용한 다각도 차트 분석

### 5.1 분석 뷰 탭 구성
- [ ] **탭 1 - 공종별 분석**: 공종 선택 → 기인물/재해형태 차트
- [ ] **탭 2 - 작업명별 분석**: 공종→작업명 선택 → 기인물/재해형태 차트
- [ ] **탭 3 - 단위작업명별 분석**: 3단계 드릴다운 → 상세 차트
- [ ] **탭 4 - 기인물별 분석**: 기인물 선택 → 재해형태 분포 차트
- [ ] **탭 5 - 교차분석**: 공종×기인물 히트맵, 기인물×재해형태 히트맵

### 5.2 각 탭 공통 차트 유형
- [ ] 수평 바 차트 (건수 순위)
- [ ] 파이/도넛 차트 (비율 분포)
- [ ] 스택드 바 차트 (재해형태 구성)
- [ ] 히트맵 (교차분석)

### 5.3 필터 기능
- [ ] 공종 멀티셀렉트
- [ ] 기인물 분류 필터
- [ ] 12대 기인물 하이라이트 토글
- [ ] 위험도 순위 범위 슬라이더
- [ ] 사고건수 최소값 필터 (노이즈 제거)

### 5.4 차트 인터랙션

## 기술 상세

### 라이브러리 (CDN 기반, 설치 불필요)
| 라이브러리 | 버전 | 용도 |
|-----------|------|------|
| TailwindCSS | 3.x (CDN) | UI 스타일링 |
| Chart.js | 4.x (CDN) | 차트 렌더링 |
| chartjs-plugin-datalabels | 2.x | 차트 데이터 라벨 |
| PapaParse | 5.x (CDN) | CSV 파싱 |
| Heroicons | SVG | 아이콘 |

### 데이터 흐름
```
CSV/JSON 파일 (정적)
    ↓ fetch() + PapaParse
JavaScript 메모리 (배열/객체)
    ↓ filter-manager.js (필터 적용)
필터링된 데이터셋
    ↓ chart-manager.js (Chart.js)
차트 렌더링 + 테이블 렌더링
```

### 브라우저 호환성
- Chrome 90+, Edge 90+, Safari 15+, Firefox 90+
- 모바일: iOS Safari, Android Chrome
